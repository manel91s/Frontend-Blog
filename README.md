# Cliente de Blog en React

Este es el repositorio principal del cliente de mi blog personal en React, que consume la API REST desarrollada con Symfony. El cliente utiliza Tailwind CSS y aprovecha React Hooks y useContext para crear una experiencia de usuario interactiva.

## Introducción

Este proyecto es la interfaz de usuario para acceder y consumir los datos de mi blog personal sobre tecnología y proyectos personales, que se encuentra en constante evolución. Está construido en React y utiliza Tailwind CSS para una interfaz atractiva y moderna.

## Funcionalidades

- **Visualización de Artículos:** Los usuarios pueden explorar los artículos disponibles en el blog.
- **Búsqueda de Artículos:** Búsqueda de artículos por palabras clave.
- **Autenticación:** Iniciar sesión y cerrar sesión de usuarios.
- **Interfaz Intuitiva:** Interfaz de usuario amigable y fácil de navegar.

## Requisitos del Sistema

- Node.js 14 o superior
- npm (o yarn) para gestionar las dependencias
